# Swift Script GUI Tool

## Dependencies
* Python 3.8
* Tkinter (python3-tk)

Note: Python 3 is required but i am not sure which version of python3 is required. Python 3.6.9 over will be fine

## How to run
1. On terminal go to the directory where main.py is located
2. On terminal run command: python3 main.py

## Functionality
1. Highlight swift keywords(limited, check keywords.py)
2. Location description of errors clickable
3. Indications(is script running and return code of last run) are located at the bottom
4. Live output of the script as it executes
5. Errors will be displayed from the execution if the script couldn't be interpreted
